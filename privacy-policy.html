隱私權政策

最後更新日期：2025年8月30日

1. 我們收集的資訊

當您遊玩我們的 Facebook Instant Game (交易技巧模擬學習挑戰) 時，我們會透過 Facebook 的 Instant Games SDK 存取以下類型的資料：

  • 玩家 ID (Player ID): 一個與您 Facebook 帳號相關聯的專屬匿名 ID，用於識別您的遊戲身份。
  • 遊戲資料: 您的遊戲進度、分數、成就以及在遊戲中的其他活動記錄。

我們不會存取或儲存您的姓名、電子郵件地址或任何其他可直接識別您個人身份的資訊。

2. 我們如何使用您的資訊

我們使用所收集的資訊來：

  • 儲存您的遊戲進度，讓您下次可以繼續遊玩。
  • 提供遊戲功能。
  • 改善遊戲體驗。

3. 資料儲存與安全

您所有的遊戲資料都安全地儲存在 Facebook 的伺服器上。

4. 資料分享

我們不會將您的任何個人資料或遊戲資料分享給第三方。

5. 資料刪除

如需刪除您的遊戲資料，請參考我們的資料刪除指示頁面。

6. 聯絡我們

如果您對本隱私權政策有任何疑問，請透過 [您的聯絡Email] 與我們聯絡。
